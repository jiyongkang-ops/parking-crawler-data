# parking-crawler-data

駐車場主要プレイヤー（タイムズ・三井のリパーク・ザ・パーク・名鉄協商・NPC・ナビパーク・エコロパーク）の
公開料金情報を、節度をもって定期収集するクローラと、その収集データ（JSONL）。

- 収集は GitHub Actions で1日3回（JST 9/15/21時）。同一物件の再訪間隔は概ね1〜3日。
- 料金は原文のまま保存（正規化しない）。変更履歴は追記型で全件保持。
- 節度ある収集5原則: 公開ページのみ / 低頻度 / 直列・待機つき / robots尊重 / 目的限定。

## 構成

- `src/` — 事業者別クローラ（`node src/run.js` ほか）
- `data/prices*.jsonl` — 料金スナップショット（追記型・`changedFromPrev` で変更検知）
- `data/*-crawl-state.json` ほか — ローリング巡回の状態
- `.github/workflows/crawl-*.yml` — 定期実行

※ 分析・レポート生成は別リポジトリ（private）で行い、本リポジトリのデータを参照する。
